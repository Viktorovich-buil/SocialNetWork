1. Очистить папку от папки .git если есть
2. Создать репозиторий на Git
3. В папке с проектом Git Bash Here (вызываем консоль)
4. git init //Инициализируйте локальный каталог как репозиторий Git
5. git config --global core.autocrlf true //связано с окнчаниями в винде и линукс
6. git add . //добавляем файлы под контроль GIT
7. git commit -m "initial commit" // Зафиксируйте файлы, которые вы разместили в локальном репозитории
8. git remote add origin https://github.com/Viktorovich-buil/SocialNetWork_SHVV.git // путь репозиторий на Git В командной строке добавьте URL для удаленного репозитория, куда будет помещен ваш локальный репозиторий
9.  git push origin master -f //добавляем файлы на сервер Git


gitk --all&     (открыть графическую оболочку гита, чтобы позырить все коммиты)
ctrl + insert (копировать в буфер обмена) 
shift + insert (вставить из буфера обмена) 
git checkout commit-number (переключится на нужный коммит) (или название файла чтобы откатить изменения)
